Corpus Slayer

An open-source corpus processing platform.

  • Target audiences: language researchers, language teachers, language students, translators;
  • Ambition: provide a feature set near to what is found on paid tools, either by developing modules that have similar functionality or gathering free-of-charge standalone tools in a single, flexible and easy-to-use interface;
  • Server: Ubuntu Linux 64-bit 16.04 LTS, on a VPS SSD2 from OVH.
